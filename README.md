# TripleTen_projects
Projects I worked on while attending the Data Analysis TripleTen Bootcamp.


| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | Americans-are-spending-more| Analysis of Personal Saving Rate, Real Disposable Income, Personal Consumption Expenditure, Inflation Rate and Credit Card Delinquency Rates|
| 2 |Exploratory-Data-Analysis-Manhattan-Airbnb-Market| Analyzed Manhattan Airbnb data to identify the most attractive neighborhoods and property types for vacation rental investment, using rental activity and revenue as key performance indicators.|
| 3 | E-commerce Conversion Funnel & Cohort Analysis | The project task was to analize raw user events logs to transform transactional data into actionable business metrics, evaluating conversion performance and customer retention over time.|
| 4 | Superstore Returns Root Cause Analysis Dashboard | Investigated the drivers of high return rates at a retail Superstore by analyzing sales, customer behavior, product categories, geography, and seasonality to support executive decision-making |
